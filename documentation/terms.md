# Terms and there meaning
## Selections (icon select_all)
This section will explain what we mean by curtain selections of addresses.
### HousingStock (icon Corporate_Fare)
All the addresses that belong to the same corporation or owner 
### Residential Area (icon View_Cozy)
This is the representation of the Dutch term  for "Wijk"
### Block (icon Format_List_Numbered_Rtl)
This represents the Dutch term for "Complex". A composition of several buildings that belong together.
### Building Type (icon Layers)
For each building design which has specific technical drawings. ("Gebouw type")
### VTW (Standardized housing types) (icon )
This is a list of building types that are more generic and though by The Association of Supervisors in Housing Corporations
### Planning Type (icon View_Timeline)
This is the same for all the houses that are physically linked together. So, for example the house 2, 4, 6, 8, 10 that are in the same row.
### Living Type (icon )
The type of building that is being occupied.
